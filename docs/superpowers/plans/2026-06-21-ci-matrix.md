### Task 1: ruff config + fix lint
- [ ] Add to `pyproject.toml` `dev` extra: `"ruff"`, `"pytest-cov"`.
- [ ] Add a conservative ruff config to `pyproject.toml`:
```toml
[tool.ruff]
target-version = "py312"
line-length = 100

[tool.ruff.lint]
select = ["E", "F", "I", "UP", "B"]
ignore = ["E501"]   # long lines tolerated (rendering/code has wide literals)
```
- [ ] Reinstall dev extra so ruff is available: `.venv/bin/pip install -e '.[dev]'`.
- [ ] Run `.venv/bin/ruff check src tests`. Fix everything it reports (imports, unused, etc.). If a rule is too noisy for this codebase, add it to `ignore` with a short comment rather than mass-editing. Do NOT change runtime behavior — lint fixes only (unused imports, import order, `f`-strings, etc.).
- [ ] Verify: `.venv/bin/ruff check src tests` → clean; `.venv/bin/python -m pytest -q` → all pass.
- [ ] Commit: `chore(lint): add ruff config and fix lint findings`.
- [ ] `roborev show <sha>`; fix any finding.

### Task 2: CI workflow matrix + coverage
- [ ] Replace `.github/workflows/ci.yml` with:
```yaml
name: ci
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    strategy:
      fail-fast: false
      matrix:
        python-version: ["3.12", "3.13"]
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: ${{ matrix.python-version }}
      - run: pip install -e ".[dev]"
      - run: ruff check src tests
      - run: python -m pytest -v --cov=herdeck --cov-report=term-missing
```
- [ ] Sanity locally: `.venv/bin/python -m pytest --cov=herdeck --cov-report=term-missing -q` runs and passes (pytest-cov installed).
- [ ] Commit: `ci: test on py3.12+3.13, run ruff and coverage`.
- [ ] `roborev show <sha>`; fix any finding.

### Done
